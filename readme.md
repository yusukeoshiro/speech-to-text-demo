## Setting up TypeScript Project


```
git clone git@github.com:metrolyinc/typescript-project-starting-point.git

cd typescript-project-starting-point

npm install --save-dev typescript ts-node dotenv prettier eslint @istanbuljs/nyc-config-typescript @types/chai @types/chai-as-promised  \
  @types/mocha @types/sinon @typescript-eslint/eslint-plugin @typescript-eslint/parser chai chai-as-promised \
  codecov dotenv eslint eslint-config-prettier eslint-plugin-prettier mocha npm-run-all nyc rimraf sinon tsc-watch


npm run dev:watch
```
